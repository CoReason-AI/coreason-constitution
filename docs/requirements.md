# Requirements

## System Requirements

*   **Python:** >= 3.12, < 3.15
*   **Operating System:** OS Independent (Linux, macOS, Windows)

## Python Dependencies

The following packages are required for the core library and server functionality:

*   **`loguru`**: For structured logging.
*   **`pydantic`**: For data validation and schema definitions.
*   **`fastapi`**: For the Governance Microservice API.
*   **`uvicorn`**: ASGI server for running the microservice.

## Development Dependencies

For contributing to the project, the following additional tools are used:

*   **`poetry`**: For dependency management.
*   **`pytest`**: For testing.
*   **`ruff`**: For code linting and formatting.
*   **`httpx`**: For testing the API server.
*   **`mkdocs`**: For documentation generation.
*   **`mypy`**: For static type checking.
